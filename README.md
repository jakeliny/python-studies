# Python Studies


## Install
Install Python:

`sudo apt install python`

Install pip:

`sudo apt install python3-pip`

Install pygeocoder:

`pip install pygeocoder`

Install serial
`pip install serial`

## Execute
`python3 arquivo.py`

On ubuntu python3 default exist
